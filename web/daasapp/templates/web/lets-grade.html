ISA Project 5 Let's Grade

  • Navigating

    Users

    Click on 'User Profiles' on the home page to access user page. We currently have 200 users on the site. Find them by putting a monotically increasing user id number at the end of the url. e.g /user/64. If you request a user id above the number of fixtures we have installed, you will be redirected to a custom 404 error page.

    Click here to take you to User Profiles.

    Create a new user - You are able to create a new user through the login page! Registering creates a new user in the db.

    Click here to create your account.

    Products

    Click on 'woa, c00l dr0n35' on the home page to access product page. We currently have 200 drones on the site. Find them by putting a monotically increasing user id number at the end of the url. e.g /product-details/64. If you request a drone id above the number of fixtures we have installed, you will be redirected to a custom 404 error page.

    Click here to take you to Product Page.

    Listings

    To create a listing, you must first log in. Then navigate to create-listing where you can enter information about your listing. This creates a new listing as well as a new drone under the product-details page.

    Click here to create a new listing (make sure you're logged in!).

    Viewing a listing - We currently have 200 listings in our db. Find them by putting a listing id number at the end of the url. ex. ../listing/64. If you request a listing id above the number of fixtures we have installed, you will be redirected to a custon 404 error page. If you created a new listing, you will be able to see both that listing and the new drone.

    Click here to take you to the Listings Page.

    Search

    You can now search for listings using the upper right hand search box!

    Project 3

    1. As a user, I would like to be able to create a user profile.

    2. As a user, I would like to be able to retrieve my own information and see other user's profiles.

    3. As a user, I would like to get information on a drone.

    4. As a user, I would like to have a biography section to express myself.

    5. As a user, I would like to show people how long I've been on the site.

    6. As a user, I would like to see the three most recent drones available.
